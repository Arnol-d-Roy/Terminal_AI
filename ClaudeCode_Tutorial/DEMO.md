### What Makes This Interactive?

The checkboxes use **GitHub-flavored Markdown** syntax:
- `- [ ]` = Unchecked
- `- [x]` = Checked

Modern markdown viewers (VS Code, GitHub, Obsidian, etc.) make these clickable!
