## Complete Workflow Example

```bash
# 1. Start the tutorial
python claude-tutorial.py

# 2. Select a level (e.g., type: 1)
# Main Menu -> Type: 1

# 3. Level menu appears with options:
#    R - Read the tutorial
#    T - Toggle task completion
#    L - List all tasks
#    S - Show stats
#    B - Back

# 4. Read the tutorial (type: R)
# The tutorial displays in your terminal

# 5. After reading, mark tasks complete
# Script asks: "Mark related tasks as complete?"
# Type task numbers: 1 2 3 4
# Tasks marked! XP earned!

# 6. Continue learning!
```

## Troubleshooting

### "Python not found"

Try these commands:

```bash
# If you have python (not python3)
python claude-tutorial.py

# If you have python3
python3 claude-tutorial.py

# Universal launcher (auto-detects)
./start-tutorial.sh
```


### "File not found" errors

Make sure you're in the tutorial directory:

```bash
cd "/mnt/d/Github Repos/Terminal_AI/ClaudeCode_Tutorial"
python claude-tutorial.py
```


### Need more help?

Run the tutorial and select **H** for help, or see
TROUBLESHOOTING.md for detailed solutions.
